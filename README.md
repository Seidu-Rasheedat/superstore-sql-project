# Retail Sales SQL Project

## Overview

This project analyzes retail sales data using SQL Server, structuring raw data into relational tables to explore trends in sales, customer behavior, and product performance.

## Data Modeling

* Created relational tables: `customers` and `orders`
* Established foreign key relationships
* Structured raw data into normalized tables

## Analysis Performed

* Sales trends by region and over time
* Customer segmentation and top customers
* Product performance and loss-making products
* Profitability analysis across regions and cities
* Impact of discounts on profit
* Shipping time analysis

## Key Insights

* A small group of customers contributes significantly to revenue
* High discount levels tend to reduce profitability
* Certain products consistently generate losses
* Sales and profit vary significantly across regions
